# Service Offering Full-Stack Workflows

## 1) Generate Project
1. Select this template in ProjectMaker.
2. Provide project metadata.
3. Cookiecutter scaffolds React frontend + Django backend.

## 2) Configure Data
1. Add initial services and testimonials (admin or seed script).
2. Validate booking API and file upload configuration.

## 3) Operate Booking Flow
1. Visitors submit reservations from public homepage.
2. Admin signs in and opens `/service-admin`.
3. Admin confirms or rejects reservations.

## 4) Iterate with Agents
1. Add backlog items for copy/design/automation improvements.
2. Route tasks to developer agent.
3. Keep docs in sync for UI and workflow changes.
