# Service Offering Full-Stack Cookiecutter

Full-stack template for service businesses and personal brands.

## Stack
- Frontend: React + TypeScript
- Backend: Django REST Framework
- DB/Cache: PostgreSQL + Redis
- Infra: Docker + ProjectMaker deployment config

## Built-in Feature Slice
- Public service homepage with testimonials and booking form
- Booking API with date/time + transaction reference + verification file
- Authenticated admin page to review/confirm/reject reservations

## Generate
```bash
cookiecutter /path/to/ServiceOfferingBookingStaticCookieCutter
```
